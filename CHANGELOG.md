# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-20

### Added
- Initial release of MarkFlow Markdown Editor.
- High-performance synced scrolling between the editor and preview.
- Complete GFM (GitHub Flavored Markdown) support via `remark-gfm` and `react-markdown`.
- Mermaid.js integration for rendering charts and flow diagrams directly from markdown blocks.
- Seamless dual-pane interface with responsive design that gracefully degrades to tabbed views on mobile devices.
- Multi-theme engine supporting Light, Dark, and a unique Cyberpunk Slate mode.
- Export pipeline providing instant Markdown (`.md`) downloads and Print-to-PDF functionality via `html2pdf.js`.
- Configured 100% Type-Safe TypeScript foundation with `eslint` optimizations.
- Minimalist open-source branding.
