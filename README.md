# MarkFlow

A modern, high-performance, distraction-free Markdown Editor and Renderer. Built with React and TypeScript.

![MarkFlow](https://via.placeholder.com/800x400?text=MarkFlow+Preview)

## Features

- **Live Data Synchronization:** Split-pane editor with live synchronized scrolling.
- **High-Fidelity Rendering:** Full GFM Markdown support with zero rendering delay.
- **Graph Visualization:** Built-in Mermaid.js diagrams directly within your Markdown workflow (```mermaid).
- **Multiple Color Themes:** Light, Dark, and Cyberpunk Slate themes included.
- **Import/Export Pipeline:** Import `.md` and `.txt` files directly, or instantly export as Raw `.md` or formatted `.pdf`.
- **Responsive Layout:** Automatically scales from dual-sidebar workstation setups into a bottom-nav tabbed interface on mobile.

## Technology Stack

- **Framework:** React + Vite
- **Language:** TypeScript 100% Type-Safe
- **Styles:** Tailwind CSS v4 
- **Icons:** Lucide React
- **Syntax Parsing:** remark-gfm, rehype-raw
- **Visualization:** Mermaid.js

## Scripts

- \`npm run dev\` - Starts the development server.
- \`npm run build\` - Compiles the app with full TypeScript type-checking.
- \`npm run typecheck\` - Uses \`tsc\` to run strict type checking.
- \`npm run lint\` - Uses \`eslint\` to lint the React and TypeScript files.
